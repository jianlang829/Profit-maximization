线性规划问题：酒厂生产优化

📌 题目描述

一家酒厂生产两种酒：低端酒和高端酒。

  • 每瓶低端酒消耗 8kg 低纯度原料酒 + 4kg 高纯度原料酒，利润 70 元。
  • 每瓶高端酒消耗 4kg 低纯度原料酒 + 7kg 高纯度原料酒，利润 120 元。
  • 每天可用原料：低纯度 360kg，高纯度 200kg。

目标：如何安排每日生产计划，使总利润最大？

数学模型：

设生产低端酒 x 瓶，高端酒 y 瓶。

目标函数： 最大化 w = 70x + 120y

约束条件：

4x + 7y ≤ 200  （高纯度原料约束）
8x + 4y ≤ 360  （低纯度原料约束）
x ≥ 0, y ≥ 0   （非负约束）

📐 解法一：找交点法（理论最优解法）

根据线性规划基本定理，最优解必定出现在可行域的顶点（即约束直线的交点）上。

步骤：

  1. 列出所有约束边界线的交点。
  2. 筛选出满足所有约束的可行点。
  3. 计算每个可行点的目标函数值，取最大者。

关键交点计算：

解方程组（两个约束同时取等号）：

2x + y = 90   （由 8x + 4y = 360 化简）
4x + 7y = 200

解得：x = 43, y = 4

利润：w = 70×43 + 120×4 = 3490 元

验证资源：

  • 低纯度：8×43 + 4×4 = 360kg （用尽）
  • 高纯度：4×43 + 7×4 = 200kg （用尽）

💻 解法二：暴力枚举法（可靠直观解法）

根据约束推导，x 的取值范围为 0 到 38。遍历所有可能的 x 值，对每个 x 计算 y 的最大可行整数值，然后比较利润。

核心算法逻辑（Python 伪代码）：

max_profit = 0
best_x, best_y = 0, 0

for x in range(0, 39):  # x 从 0 到 38
    # 根据约束计算 y 的最大值
    y_from_high = (200 - 4 * x) // 7  # 高纯度约束
    y_from_low = (360 - 8 * x) // 4   # 低纯度约束
    y = min(y_from_high, y_from_low)  # 取更严格的约束
    if y < 0: y = 0                   # 保证非负

    profit = 70 * x + 120 * y

    if profit > max_profit:
        max_profit = profit
        best_x, best_y = x, y

print(f"最优解: x={best_x}, y={best_y}, 利润={max_profit}")

✅ 最终结论：两种方法均得出相同最优解
生产 43 瓶低端酒 和 4 瓶高端酒，可获得最大日利润 3490 元。
